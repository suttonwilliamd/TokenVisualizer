## üë©‚Äçüíª Getting Started

### Prerequisites

- Node.js v14+  
- npm (or yarn)

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/token-viewer.git
   cd token-viewer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the dev server**
   ```bash
   npm run dev
   ```

4. **Open** [http://localhost:4321](http://localhost:4321) in your browser.
